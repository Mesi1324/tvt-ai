### Overview
This is a Retrieval-Augmented Generation (RAG) application designed for the **FDRE Technical and Vocational Training (TVT) Institute**. It allows users to query PDF documents (Curricula, Standards, Manuals) using natural language.

### Features
- **Multilingual:** Supports English and Amharic queries.
- **Cost:** Free to deploy and use.
- **Privacy:** Document processing happens in-session.

### How to Run Locally
1. Clone the repo.
2. Run `pip install -r requirements.txt`.
3. Run `streamlit run app.py`.

### Deployment
This app is designed for **Streamlit Community Cloud**. Please add your `GOOGLE_API_KEY` in the Streamlit Secrets management console.
